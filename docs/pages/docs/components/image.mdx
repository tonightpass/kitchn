# Image

Display image content.

## Usage

First of all, you need to import the `Image` component from the `kitchn` package.

```js
import { Image } from "kitchn"
```

## Static

<Playground
  code={`
  <Image
    src={"https://picsum.photos/540/300"}
    alt={"Lorem picsum image"}
    width={540}
    height={"auto"}
    htmlWidth={540}
    htmlHeight={300}
/>
`}
/>

## Props

| Name | Type | Default | Required | Description | Accepted values |
| :--- | :--- | :-----: | :------: | :---------- | :-------------- |
| `src` | `string` | `undefined` | ✅ | The URL or path of the image. | - |
| `alt` | `string` | `undefined` | ✅ | The alternative text for the image. | - |
