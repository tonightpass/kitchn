# Checkbox

Displays a boolean value.

## Usage

First of all, you need to import the `Checkbox` component and the `useCheckbox` hook from the `@tonightpass/kitchen` package.

```js
import { Checkbox, useCheckbox } from "@tonightpass/kitchen"
```

## Default

<Playground
code={`() => {
  const [checked, toggle] = useCheckbox();

  return (
    <Checkbox checked={checked} onChange={toggle}>
      Option
    </Checkbox>
  );
}`}
/>

## Checked by default

<Playground
code={`() => {
  const [checked, toggle] = useCheckbox(true);

  return (
    <Checkbox checked={checked} onChange={toggle}>
      Option
    </Checkbox>
  );
}`}
/>

## No Label

<Playground
code={`() => {
