## Props

| Name | Type | Default | Required | Description
|------|------|---------|----------|-------------
| `type` | [`AccentColors`](/docs/theming#accent) | `'primary'` | No | Visual style variant of the badge.
| `size` | `'small' \| 'normal' \| 'large'` | `'normal'` | No | Size of the badge.
